## About / 关于

**Math Rock Essentials** is a comprehensive guitar guide covering the core techniques, theory, and creative tools for math rock guitar. Written by Stephen Hazel, this guide covers everything from tapping, fingerpicking, and hybrid picking, to music theory, chord progressions, alternate tunings, fretboard mastery, and functional strumming patterns.

This repository is its **Chinese translation**, presented bilingually alongside the original text. The project is structured to mirror the original book's three-part layout.

《Math Rock Essentials》是一本系统讲解数学摇滚吉他的全面指南，涵盖技巧、理论和创作工具。本仓库是其**中文翻译版**，原文与译文并列呈现，结构完全对标原书的三大部分。

## Compilation / 编译

Requires **XeLaTeX** with `ctex` package.

```shell
xelatex Math-Rock-Essentials.tex
```

Or use any LaTeX editor (Overleaf, VS Code + LaTeX Workshop, TeXShop, TeXstudio).

Each chapter is a separate file under `chapters/`, included into the main `.tex` via `\include{}` or `\input{}`. Edit them individually without touching the master file.

## Translation Style / 翻译规范

| Element | Style |
|---------|-------|
| **Original English** | Deep blue (`#1A2A3A`), prefixed with `▸ ENG` |
| **Chinese Translation** | Red (`#C0392B`), prefixed with `▸ 译` |
| **Terminology** | Purple (`#2774AE`), with English in parentheses |
| **Tips & Notes** | Blue info boxes (`notebox`) |
| **Examples** | Green-bordered boxes (`examplebox`) |
| **Exercises** | Dark-bordered boxes with auto-numbering |
| **Short phrases** | Side-by-side via `\enandcn{}{}` |

See the preamble of `Math-Rock-Essentials.tex` for all reusable commands: `\eng{}`, `\chn{}`, `\term{}{}`, `\enandcn{}{}`, `notebox`, `examplebox`, `exercise`.
