# Math Rock Essentials — Chinese Translation Project

This repository is a **Chinese translation** of **"Math Rock Essentials"** by [Stephen Hazel](https://www.ultimate-guitar.com/contribution/StephenHazel/).

## About the Original Work

"Math Rock Essentials" is a comprehensive guide for learning math rock guitar techniques, originally authored by Stephen Hazel. It covers a wide range of topics including strumming patterns, fingerpicking, tapping, hybrid picking, alternate tunings, and more — all essential skills for playing math rock.

## About This Project

This project aims to make the content of "Math Rock Essentials" more accessible to Chinese-speaking musicians by providing Chinese translations of the original text and materials. Whether you prefer Guitar Pro files (`.gp`, `.gp5`, `.gpx`) or PDFs, all source files are preserved alongside the translated versions.
